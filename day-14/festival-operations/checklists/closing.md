# Festival Closing Checklist

**Timing:** Begin 30 minutes before official closing time  
**Responsible Party:** Festival Operations Manager + Team Leads

---

## T-30 Minutes: Pre-Closing Preparation

### Customer Experience (Zelda)
- [ ] Announce "30 minutes until closing" (all channels)
- [ ] Stop accepting new customers in lines
- [ ] Clear existing queues gracefully
- [ ] Direct last-minute customers to faster attractions
- [ ] Thank customers for coming

### Security Team (Marcus)
- [ ] Begin perimeter sweep from back to front
- [ ] Note any lingering groups or individuals
- [ ] Prepare for guest exit flow management
- [ ] Position staff at exits for guest goodbyes
- [ ] Verify all emergency equipment accounted for

### Marketing & Communications (Elena)
- [ ] "Thanks for coming!" social media post ready
- [ ] Tomorrow's hours confirmed for posting
- [ ] Photo highlights selected for evening post
- [ ] Update website status to "Closed - See you tomorrow!"

---

## T-0: Official Closing Time

### Gates & Entry
- [ ] Close ticket sales
- [ ] Allow last guests to exit naturally (no rushing)
- [ ] Begin entry area breakdown
- [ ] Collect and secure ticket scanners
- [ ] Store signage and barriers
- [ ] Count cash/reconcile registers

### All Customer Areas
- [ ] Complete all in-progress services/readings
- [ ] Politely guide remaining guests toward exits
- [ ] Thank every guest personally
- [ ] Note any complaints or compliments

---

## Post-Closing: Vendor Operations

### Vendor Coordinator (Marcus)
- [ ] Collect remaining balance payments from all vendors
- [ ] Document any vendor issues from the day
- [ ] Note any vendor requests for tomorrow
- [ ] Verify all vendors know tomorrow's schedule
- [ ] Walk through to ensure proper breakdown
- [ ] Collect any vendor incident reports
- [ ] Schedule any needed vendor meetings

---

## Facilities & Cleanup

### Lost & Found (Maria)
- [ ] Process all items received in last hour
- [ ] Complete photo documentation for all new items
- [ ] Update master log with today's items
- [ ] Secure all high-value items in locked cabinet
- [ ] Dispose of perishables properly
- [ ] Set up for tomorrow's intake

### Facilities Team
- [ ] Complete grounds trash collection
- [ ] Empty and secure all trash receptacles
- [ ] Final restroom check and cleanup
- [ ] Turn off non-security lighting
- [ ] Secure all buildings and structures
- [ ] Check for lost items in high-traffic areas

---

## Security & Safety

### Final Security Sweep
- [ ] Complete walk of entire grounds (back to front)
- [ ] Check all buildings are locked
- [ ] Verify emergency exits secured
- [ ] Confirm all guests have departed
- [ ] Check parking areas are clear
- [ ] Note any security concerns for tomorrow

---

## Team Debrief & Planning

### End-of-Day Meeting (15 minutes)
- [ ] Share today's wins and successes
- [ ] Discuss any incidents or issues
- [ ] Review lost & found highlights
- [ ] Weather forecast for tomorrow
- [ ] Staffing changes or needs
- [ ] Special events or VIPs tomorrow

# 🚀 Gift Tag Generator - Quick Start

**Advent of AI - Day 9 Challenge**

## ⚡ 30-Second Setup

1. **Import Recipe:**
   - Open Goose → Settings → Recipes
   - Import `gift-tag-generator.yaml`

2. **Generate a Tag:**
   ```
   Create an elegant gift tag for Sarah Chen, 
   handmade scarf, medium size
   ```

3. **Print:**
   - Open generated HTML
   - Cmd+P / Ctrl+P
   - Print!

## 🎨 Style Examples

| Style | Best For |
|-------|----------|
| **Elegant** | Professional, VIP, formal |
| **Playful** | Children, casual, fun |
| **Minimalist** | Modern, design-conscious |
| **Festive** | Holidays, celebrations |

## 📏 Size Guide

- **Small** (3"×2") - Small gifts, compact
- **Medium** (4"×3") - Standard gifts
- **Large** (5"×4") - Large gifts, elaborate

## 💬 Example Commands

### Simple:
```
Make a gift tag for John
```

### Detailed:
```
Generate a playful large gift tag for Emma,
stuffed unicorn from Aunt Sarah,
include a poem, bright colors
```

### With QR Code:
```
Create an elegant tag for Dr. Chen,
stethoscope from medical board,
QR code: https://medboard.org/tribute,
formal tone, navy blue
```

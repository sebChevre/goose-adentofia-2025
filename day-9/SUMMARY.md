## 🚀 How to Use

### For End Users

1. **Import the recipe:**
   - Open Goose Desktop app
   - Go to Settings → Recipes
   - Import `gift-tag-generator.yaml`

2. **Generate a tag:**
   ```
   Create an elegant gift tag for Sarah Chen, 
   medium size, with a heartfelt poem
   ```

3. **Print:**
   - Open generated HTML in browser
   - Print with recommended settings
   - Cut to size and attach to gift

### For Developers

1. **Review the PRD** for complete specifications
2. **Examine the YAML** for implementation details
3. **Read the README** for usage patterns
4. **Extend as needed** for custom requirements

## 📊 Technical Specifications Summary

### Output Format
- **File Type:** HTML5
- **Styling:** Inline CSS only
- **Dependencies:** None (fully self-contained)
- **Print Optimization:** @page rules, print media queries
- **Color Management:** RGB with CMYK guidance

### Performance
- **Generation Time:** < 30 seconds per tag
- **File Size:** ~10-20 KB per tag
- **Browser Support:** All modern browsers
- **Print Compatibility:** All standard printers

- **CSS:** CSS3 with print optimizations

- **Internationalization:** UTF-8 encoding, proper lang attributes
